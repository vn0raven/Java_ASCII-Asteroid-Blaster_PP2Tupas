# Demo Script / Presentation Guide

Good day everyone. Our project is called ASCII Asteroid Blaster, a Java Swing retro arcade game built to demonstrate the four pillars of Object-Oriented Programming.

During the demo, we will run the game live and show the player movement, directional shooting, asteroid enemies, meteor shard hazards, UFO enemies, scoring, upgrades, sound effects, and game states. The player can move in eight directions and shoot based on the ship's facing direction. Asteroids and hazards appear in the arena, and the player must shoot or dodge while earning points.

The project demonstrates encapsulation through separate classes for player data, scoring, input, asteroids, and upgrades. It demonstrates inheritance through shared object classes such as `GameObject`, `Asteroid`, and `PowerUp`. It demonstrates polymorphism because different objects can be updated and drawn through shared behavior. It demonstrates abstraction through interfaces and abstract parent classes such as `Updatable`, `Drawable`, and `GameObject`.

Overall, the game is a complete playable OOP application with source code, UML diagrams, documentation, screenshots/proof folder, audio files, and run scripts.
